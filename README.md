# djasa2288.github.io

Is a Simple My GitHub Pages Site

Welcome to my GitHub Pages site! This site is built using [Jekyll](https://jekyllrb.com/) and the [Just the Docs](https://pmarsceill.github.io/just-the-docs/) theme.

## About

This site serves as a personal website and blog where I share my projects, tutorials, and other content. It features a clean and simple design with a navigation sidebar for easy access to different sections.

## Features

- **Responsive Design:** The site is mobile-friendly and looks great on all devices.
- **Navigation Sidebar:** Easy navigation with a sidebar that collapses on smaller screens.
- **Markdown Support:** Write content using Markdown for simplicity and readability.
- **Customizable:** Easily change the theme and layout to suit your needs.

## Structure

The site is organized as follows:

- `index.md`: The homepage of the site.
- `_config.yml`: The configuration file for Jekyll and the Just the Docs theme.
- `about.md`: An example of an additional page.

## Getting Started

To run this site locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/username/username.github.io.git
   cd username.github.io
   ```

2. **Install Jekyll and dependencies:**
   Make sure you have Ruby and Bundler installed. Then run:
 
   ```bash
   bundle install
   ```

3. **Run the site locally:**
   ```bash
   bundle exec jekyll serve
   ```
   Open your browser and go to `http://localhost:4000` to see the site.

## Customization

To customize the site, you can edit the `_config.yml` file and the Markdown files in the root directory. For more information on how to customize the Just the Docs theme, refer to the [official documentation](https://pmarsceill.github.io/just-the-docs/).
